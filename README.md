SaddiquePortfolio 

A modern, multipage portfolio template for developers. Fully responsive, fast, and optimized for showcasing projects.

Easy to setup 

Free & OpenSource 

Multi-page & responsive 

Fast & SEO optimized 

Supports custom theme colors 

Technologies

HTML | CSS | JavaScript | SASS

Quick Start
# Clone repository
$ git clone https://github.com/YOURUSERNAME/saddiqueportfolio
$ cd saddiquefolio

# Install dependencies
$ npm install

# Compile SASS to CSS
$ npm run compile:scss


Open index.html in your browser or use Live Server.

Customization

Change theme color: sass/abstracts/_variables.scss → $themeClrPrimary

Update content: /index.html → Hero, About, Projects, Contact

Add projects: Duplicate .projects__row in Projects section
